EldoS Keeper source code.

EldoS Keeper is built using exactly this code. 
Note, that if you get an Access Violation, this doesn't mean, that there is a
problem with some code. This can happen because of the compiler options.

This demo uses freeware third-party component named HexEditor.

This code can be compiled only with C++Builder 4 or 5. C++Builder 3 can't be
used.
